A lightweight and customizable "Scroll to Top" button for WordPress websites. Improve your site's user experience with smooth scrolling.

== Description ==

Shohel Scroll To Top is a simple and user-friendly WordPress plugin that adds a "Back to Top" button to your website. This improves the navigation experience, especially for long pages.

**Features:**
- Lightweight and optimized for speed
- Customizable button position and style
- Smooth scroll animation
- Works on all modern themes

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Done! The scroll-to-top button will appear on your site.
